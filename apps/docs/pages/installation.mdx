## Installation

This section will show you how to run East PSBT UI Builder locally using the regtest network that have been installed in the previous section.
1. Clone the repo
```bash
git clone https://github.com/EAST-Blue/east-bitcoin-lib.git & cd east-bitcoin-lib
```

2. Build and run regbox
```bash
yarn workspace @east-bitcoin-lib/regbox build && node ./packages/regbox/dist/index.js
```

3. Run this command and it should show the balance of the default wallet
```bash
bitcoin-cli -regtest -rpcuser=east -rpcpassword=east getbalance
```
