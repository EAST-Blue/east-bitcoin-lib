# Introduction
 
Welcome to East PSBT UI Builder docs! This doc will show you how to setup the project in your local machine.

## What is East PSBT UI Builder?

**East PSBT UI Builder** is a web application to show developers on how to use the [east-bitcoin-lib](https://github.com/EAST-Blue/east-bitcoin-lib), an sdk to build Bitcoin PSBT in a simple way.
