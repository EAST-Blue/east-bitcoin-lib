# East SDK

> East SDK: Easy Bitcoin Transactions and Metadata Indexing.

East SDK consists of 2 SDKs: 
1. Smart Index SDK.
This SDK is utilized to index transaction metadata within the Eastnode protocol, enabling efficient organization and retrieval of transaction information for various applications and processes.

2. Bitcoin SDK.
This software development kit will be primarily utilized to create transactions in Bitcoin using Partially Signed Bitcoin Transactions (PSBT).
