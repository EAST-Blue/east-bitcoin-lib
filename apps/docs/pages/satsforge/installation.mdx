## Installation

This section will show you how to run Satsforge locally using the regtest network that have been installed in the previous section.

1. Clone the repo

```bash copy
git clone https://github.com/EAST-Blue/east-bitcoin-lib.git && cd east-bitcoin-lib/apps/ui
```

2. Build the project

```bash copy
yarn build
```

3. Start the project

```bash copy
yarn start
```

4. Go to **[localhost:3001/psbt](http://localhost:3000)**
