# Using local Regbox

There are number of reasons why developing locally is preferred (more customizable, confidential reasons, etc). If you intend to utilize RegNet on your personal machine, you should use the regbox. Ensure that **Docker** is installed on your machine beforehand.

## Installing regbox

```bash copy
npm i -g @east-bitcoin-lib/regbox
```

## Running regbox

```bash copy
regbox start
```
