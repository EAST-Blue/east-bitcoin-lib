# Introduction
 
Welcome to East PSBT UI Builder docs! This doc will show you how to setup the project in your local machine.

## What is East PSBT UI Builder?

**East PSBT UI Builder** is a web application to show developers on how to use the [east-bitcoin-lib](https://github.com/EAST-Blue/east-bitcoin-lib), an sdk to build Bitcoin PSBT in a simple way.

## Installation

This section will show you how to run East PSBT UI Builder locally using the regtest network that have been installed in the previous section.
1. Clone the repo
```bash
git clone https://github.com/EAST-Blue/east-bitcoin-lib.git && cd east-bitcoin-lib/apps/ui
```

2. Install dependencies
```bash
yarn
```

3. Run project
```bash
yarn dev
```

4. Go to **[localhost:3001/psbt](http://localhost:3001/psbt)**
